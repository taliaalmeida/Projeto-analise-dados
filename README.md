# 📊 Projeto de Análise de Dados — Cancelamento de Clientes

Projeto desenvolvido durante curso de Python com foco em análise de dados e visualização.

## 📌 Sobre o Projeto

Análise completa do cancelamento de clientes (churn) de uma empresa, com criação de gráficos e dashboards para identificar as principais causas e propor ações de melhoria.

## ✅ Resultado

- Identificadas as principais causas de cancelamento
- Propostas ações de melhoria baseadas nos dados
- Taxa de cancelamento reduzida de 56% para 18%

## 🛠️ Tecnologias Utilizadas

- Python
- Pandas
- Jupyter Notebook
